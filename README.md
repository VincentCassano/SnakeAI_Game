## 📋 项目介绍
这是一款精心设计的贪吃蛇游戏升级版，融合了智能AI对抗系统和多种游戏模式。游戏采用Python开发，基于Pygame库构建了精美的图形界面，提供了丰富的游戏体验、灵活的自定义选项和沉浸式的音效反馈系统。无论是休闲玩家还是想挑战AI的玩家，都能在这里找到乐趣。

## ✨ 游戏特性

### 🎮 多样的游戏模式
- **普通模式**：经典的贪吃蛇玩法，控制蛇收集食物并不断变长，挑战最高分
- **对抗模式**：与AI控制的蛇进行实时对战，看谁能生存更久或获得更高分数
- **限时模式**：在60秒内进行快速挑战，紧张刺激，争取获得最高分数

### 🧠 智能AI系统
- 基于先进路径规划和决策算法的AI对手
- 可调节的AI难度级别（简单、中等、困难），适应不同玩家水平
- AI能够感知环境、规划路径并做出战略性移动，提供真实的对战体验

### 🎨 精美的视觉效果
- 渐变色背景设计，提供舒适的视觉体验
- 平滑的蛇身移动和动画效果，增强游戏流畅度
- 直观友好的UI界面，包括游戏菜单、设置面板和帮助界面
- 全面支持中文字体显示，确保游戏内文字清晰可读

### 🔊 沉浸式音效反馈
- 食物获取音效，提供即时反馈
- 倒计时音效，增加游戏紧张感
- 游戏结束音效，增强游戏体验
- 可在设置中灵活开启或关闭音效

### ⚙️ 丰富的自定义选项
- **棋盘大小**：自由调整游戏棋盘的尺寸，适应不同屏幕
- **单元格大小**：调整每个网格单元格的像素大小，优化视觉效果
- **蛇速调节**：自定义蛇的移动速度，从休闲到极速挑战
- **食物价值**：设置每个食物的基础分数，改变游戏策略
- **边框设置**：可开关游戏边界碰撞检测，体验无边界模式
- **AI难度**：灵活设置AI对手的难度级别，适合从初学者到高手

### 📖 详细的游戏帮助
- 完整的操作指南，快速上手
- 清晰的游戏规则说明，了解所有玩法
- 实用的快捷键提示，提高操作效率
- 模式选择帮助，选择最适合的游戏体验

## 📦 安装说明

### 💻 系统要求
- **Python版本**：3.6或更高版本
- **操作系统**：支持Pygame的操作系统（Windows, macOS, Linux）
- **硬件要求**：基本的图形显示能力和处理性能
- **音频设备**：可选，用于体验游戏音效

### 🚀 快速开始
1. **克隆或下载仓库**
   ```bash
   git clone https://github.com/VincentCassano/SnakeAI_Game.git
   cd snakeAI_Game
   ```

2. **安装依赖包**
   ```bash
   pip install -r requirements.txt
   pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
   ```

3. **启动游戏**
   ```bash
   python snakeAI_Game-v1.0.6.py
   ```

## 🎯 使用方法

### 🏠 主菜单操作
游戏启动后，将直接进入图形界面的主菜单，您可以：
- 按下**数字键1**：进入普通模式
- 按下**数字键2**：进入对抗模式（与AI对战）
- 按下**数字键3**：进入限时模式
- 按下**数字键5**：打开游戏设置面板
- 按下**数字键6**：查看游戏帮助与教程
- 按下**ESC键**或**q键**：退出游戏

### 🎮 游戏控制
在游戏过程中，您可以使用以下按键控制蛇的移动：
- **方向键**（上、下、左、右）：控制蛇的移动方向
- **WASD键**：另一种控制方式，同样控制蛇的移动方向
- **ESC键**：暂停或继续游戏

## 📜 游戏规则详解

### 🎯 基本规则
1. 控制贪吃蛇在棋盘上移动，吃掉随机出现的食物
2. 每吃掉一个食物，蛇的身体会变长一节，同时获得相应分数
3. 蛇不能撞到游戏边界（除非在设置中关闭了边框）
4. 蛇不能撞到自己的身体
5. 撞到边界或自己身体会导致游戏结束

### 🆚 模式特定规则

#### 对抗模式
- 您和AI控制的蛇同时在棋盘上移动
- 您可以通过吃到食物或让AI蛇撞到障碍物来获得优势
- 最后存活的一方获胜，或者在双方同时存活的情况下，得分更高者获胜

#### 限时模式
- 游戏倒计时为60秒
- 在时间结束前尽可能获得更多分数
- 随着分数增加，蛇的移动速度会逐渐加快，增加挑战性
- 时间结束后，显示最终得分和排名

#### 影子模式
- AI蛇完全模仿玩家的移动和蛇身长度，AI就是影子
- AI死亡玩家也会死亡游戏倒计时为60秒
- 随着分数增加，蛇的移动速度会逐渐加快，增加挑战性
- 无胜利条件


## ⚙️ 游戏设置指南
在设置界面中，您可以调整以下选项：

1. **棋盘大小**：调整游戏棋盘的大小（默认40）
2. **单元格大小**：调整每个网格单元格的像素大小（默认20）
3. **蛇速**：调整蛇的移动速度，值越小速度越快（默认0.10秒/步）
4. **食物价值**：设置每个食物的基础分数（默认1）
5. **开启音效**：切换游戏音效的开启/关闭状态（默认开启）
6. **开启边框**：切换游戏边界碰撞检测的开启/关闭状态（默认开启）
7. **AI难度**：设置AI对手的难度级别（简单、中等、困难）（默认中等）

**注意**：部分设置需要重新开始游戏才能生效。
